First Order Form

Converting a System to First Order Form

So what do we mean by first order form for a systems of ODEs? A system of ODEs can be written in a few different ways that are equivalent. First order form is a way of representing the system of ODEs in which there are no derivatives higher than a first derivative. An example of an ODE that is NOT is first order form is:

(1) ${d^4x_1}/{dt^4} + a{d^3x_1}/{dt^3} + b {d^2x_1}/{dt^2} + c {dx_1}/{dt} + d x_1 = 0$

Converting this to first order form is easy. First you identify the dependent variables. These are the variables that appear in the numerator of the derivates terms. For each dependent variable, you find the highest order derivative. In this case, the only dependent variables is $x_1$ and the highest order derivative is ${d^4x}/{dt^4}$. We define a new variable name for each derivative that is less than the highest order derivative for each dependent variable. However as we do this, we start with the first derivative and work our way up as shown below:

(2) ${dx_1}/{dt} = x_2$
(3) ${dx_2}/{dt} = x_3$
(4) ${dx_3}/{dt} = x_4$

Note that equation (3) and (4) are the same as ${d^2x}/{dt^2}$ and ${d^3x}/{dt^3}$ respectively, but are written as first order derivatives. We now replace ${d^4x}/{dt^4}$ in (1) with the first order derivative ${dx_4}/{dt}$, and replace the lower order derivatives in (1) with the variables $x_2$, $x_3$, and $x_4$. For first order form, it is common to put the first order derivative term alone on the left hand side of the equation and move all other terms to the right hand side of the equation. Equation (1), in first order form, is equations (2), (3), and (4) combined with equation (1) with the substitutions mentioned above:

${dx_1}/{dt} = x_2$
${dx_2}/{dt} = x_3$
${dx_3}/{dt} = x_4$
${dx_4}/{dt} = -a x_4 - b x_3 - c x_2 - d x_1$

First Order Form for Matrix Equations

A matrix equation with higher order derivatives can also be converted to first order form fairly easily. As an example, lets consider a matrix equation like the following:

(5) ${d^3x_i}/{dt^3} + B_{ij} {d^2x_i}/{dt^2} + C_{ij} {dx_i}/{dt} + D_{ij} x_i = 0$

where $x_i$ is the vector:

$x_i = |\table x_1;x_2;...;x_n|$

In this case, the highest order derivative is third order. There are $n$ variables, and for each variable, we define a new variables for each of the first order derivative:

${dx_1}/{dt} = x_{n+1}$
${dx_2}/{dt} = x_{n+2}$
...
${dx_n}/{dt} = x_{2n}$

We also define a new variable for each of the second order derivatives:

${dx_{n+1}}/{dt} = x_{2n+1} = {d^2x_1}/{dt^2}$
${dx_{n+2}}/{dt} = x_{2n+2} = {d^2x_2}/{dt^2}$
...
${dx_{2n}}/{dt} = x_{3n} = {d^2x_n}/{dt^2}$

Now we append these new variables to the vector $x$:

$x = |\table x_1;x_2;...;x_{3n}|$

If $0$ is an n x n matrix of 0s, and $I$ is an n x n identity matrix, then matrix equation (5) can then be written as:

${dx}/{dt} = (\table 0, I, 0; 0, 0, I; -D, -C, -B) x$
