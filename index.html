 Generalizing Modal Analysis - Introduction
Lance Larsen
Modal analysis is a powerful tool for analysis of linear ordinary differential equations (ODEs). Modal analysis involves finding a set of modal coordinate which decouples a system of ODEs. A useful definition for a modal coordinate is the following:

Modal Coordinate definition:
A modal coordinate is a linear function that when used as a coordinate results in a decoupled ODE which can be solved independent of any other coordinates.

In essence, modal analysis converts a system that is complex to understand, because changes in one variable affect the response of other variables, to a system that is simple to understand, because the modal ODEs do not interact with one another and the modal equations are easy to solve. The modal ODE solutions can be added to get a total system response. The power of modal analysis comes from the fact that you can understand the total behavior of the system in terms individual behavior of the modes.

Calculating Modal Coordinates

Suppose we have a linear system of ODEs expressed in matrix first order form:

(1) ${dx_i}/{dt} = A_{ij} x_j$

In order for modal analysis to work, the value of the coefficients in $A_{ij}$ must not change with time. In other works, the coeffients can be constant values, or constant parameters such as a mass or stiffness quantity. A modal coordinate is a linear function, which means it can be expressed as:

(2) $q = c_i x_i$

where $c_i$ is a vector with values that do not change with time. Lets choose the coefficients $c_i$ such that $c_i$ is an eigenvector, which satisfies the equation:

(3) $c_i A_{ij} = λ c_j$

This is sometimes called a left eigenvector because it is an eigenvector calculated by multiplying the matrix by a row vector on the left. Now we left multiply the system of ODEs in (1) by the eigenvector $c_i$ and apply equation (3):

(4) $c_i {dx_i}/{dt} = c_i A_{ij} x_j = λ c_i x_i$

Since $c_i$ is a vector of constants, we can insert it inside the differential on the left hand side of the equation.

(5) ${d(c_i x_i)}/{dt} = λ c_i x_i$

Applying equation (2) we get the decoupled modal ODE:

(6) ${dq}/{dt} = λ q$

The solution of the modal equation is:

$q = C e^{λ t}$
